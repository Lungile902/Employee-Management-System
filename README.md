# Employee-Management-System
database management for employees 
### Employees at Ekuhlaleni company needed an employee database to manage their employees. The employees must be managed in every aspect contribution to the company. This includes their personal details, work they perform to the company, to their salaries. Therefore a complete database was established,with database schema, table, sql inserts, sql table, sql queries, trigers, function, and views.
### The following entities were created in order to form an accurate relational database schema for the business rules.
